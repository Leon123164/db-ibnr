# Deutsche Bahn IBNR JSON

This repository contains all Deutsche Bahn IBNR numbers in a single structured JSON file.

The data is based on the official files provided by  
https://www.michaeldittrich.de/ibnr/

Big thanks to Michael Dittrich for his great and important work.
